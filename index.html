Welcome to Editor

An ultra-minimalistic text editor that runs in the browser. What you type is autosaved to localStorage every 2 seconds, so they'll persist even if you close the tab or refresh the page.

You can also paste or drag-and-drop images on firefox

Keybinds

F1 or ctrl + ,: Toggle this menu

ctrl + d: Toggle dark theme

Settings

Hint: Try monospace or serif for the browser's default monospace/serif fonts
